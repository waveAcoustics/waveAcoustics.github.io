7. Simulation settings

7.1 Domain type

This option is used to set the domain of a simulation as a free field or, alternatively, as a half space.

The next figure depicts a simulation of sound traveling around an acoustic barrier, using the ‘Half space’ setting.

7.2 Resolution

This option sets the domain resolution, i.e. the size of each calulation cell. Changing this value will change the physical size of the simulation domain. For example, the domain size corresponding to the default resolution (0.04m) is 22m X 22m - If the resolution value is changed to 0.08m, the size of the domain will be 44m X 44m. Hence, a sine wave with a given frequency will visually appear tice as short using the 0.08m resolution than it would using 0.04m resolution.
