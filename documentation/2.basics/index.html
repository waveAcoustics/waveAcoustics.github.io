2. Basic Usage

2.1 Application layout

The next figure depicts the Wave Acoustics Simulator GUI (Graphical User Interface). It consists of four main regions:

  1. File menu - Here the user can quit the application, load a new simulation or check information about WaveAcoustics itself.

  2. Simulation options – The user can set the source and boundary characteristics, view data in real time using a virtual oscilloscope and change graphical and other simulation settings. Each option type is grouped into an expandable tab, listed below. These options are described in greater detail in the subsequent chapters of this documentation.

    • Source

    • Boundaries

    • Oscilloscope

    • View

    • Settings

  3. Main view - Show the graphical output of wave simulations.

  4. Simulation controls - Play/pause, reset and set snap grid on/off.

2.2 How to setup a basic simulation

2.2.1 Input Source

Right-click on the main view to create by default a propagating sound pulse:

If you wish to input a different type of source, use the expandable ‘Source’ menu, in the simulation options.

2.2.2 Input boundaries

Left-click on the main view. This will draw boundary lines:

Please note that whilst the boundary position has not been fixed by the user, the boundary will appear pink. Otherwise, it will appear white.

The user must press the ‘ESC’ key in order to stop inputting more boundaries.

2.2.3 Snap grid on/off

You can toggle grid snapping by clicking the snap button in the simulation controls:

It is worth noting that when inserting boundaries (and while the grid mode is enabled) a tooltip appears next to the mouse cursor indicating the corresponding physical distance of the boundary.

2.2.4 Select/Delete boundaries

The user is able to select boundaries by using SHIFT + RIGHT CLICK and dragging the mouse cursor over it. Selected boundaries will appear in yellow. The following figure shows two boundaries have been selected:

In order to unselect the boundaries, the user should press the ‘ESC’ key. It is possible to delete selected boundaries by pressing the key ‘DEL’ on the keyboard. The following figure shows the arrangement obtained by deleting the two previously selected boundaries:

2.2.5 Simulation output

The following picture depicts the graphical output of the Wave Acoustics Simulator, where the user has defined a pulse and a set of boundaries:

Note the two point responses (blue and red) visible in the ‘Oscilloscope’ tab. These responses are updated while the simulation is running. The corresponding receiver positions are set by clicking the middle mouse button in the main view area – first click sets the first receiver (green square), while the second click sets the second receiver (green circle). To reset these receiver positions press the ‘R’ key.

2.2.6 Save data for post-processing

The acoustic pressure data corresponding to the two receivers can be saved to disk by enabling ‘Save data to disk’. Then, once the option to save data in unchecked, the process of saving data is stopped and the receiver data is saved to a .csv file inside the ‘data_output’ folder. The .csv file can be opened by 3rd party software such as a spreadsheet, MATLAB or Python. As an example, the following figure shows the data being opened using a spreadsheet:

2.2.7 Reset simulation

To reset to an entirely new simulation, click ‘File -> New’ in the file menu:

To reset only the sound field and source positions, click the reset button in the simulation controls:

To reset only the receiver data and positions, press the ‘R’ key.

2.3 Keyboard shortcuts

Several actions on WAS can be performed using the keyboard. The following lists them:

  • ‘ESC’ - is used to finish entering boundaries, the boundaries become active and will appear white.

  • ‘ENTER’ - is used to close the boundaries.

  • ‘SPACE’ - Pause/Unpause the simulation

  • ‘DELETE’ - is used to remove selected boundaries from the simulation.

  • ‘R’ - Pressing the ‘R’ key resets the receivers position and data.
