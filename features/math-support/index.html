waveAcoustics uses the numerical method Finite-difference time-domain (FDTD) to compute wave fields.

The FDTD method

Finite-difference time-domain (FDTD) is a numerical method used to find an approximate solution to the linear wave equation.

In 2D, the wave equation has the following form:

\[ \frac{1}{c^2}\frac{\partial^2 {p}}{\partial {t}^2} = \frac{\partial^2 {p}}{\partial {x^2}} + \frac{\partial^2 {p}}{\partial {y^2}} \]

where p denotes sound pressure, t denotes time, x and y are the spatial coordinates.

This is a second order equation that describes the time variation of sound pressure in a two-dimensional domain. This equation is equivalent to a system of first-order equations:

\[ \frac{1}{c}\frac{\partial {p}}{\partial {t}} = \frac{\partial {v_x}}{\partial {x}} + \frac{\partial {v_y}}{\partial {y}} \]
\[ \frac{\partial {v_x}}{\partial {t}} = \frac{\partial {p}}{\partial {x}} + \frac{\partial {p}}{\partial {y}} \]
\[ \frac{\partial {v_y}}{\partial {t}} = \frac{\partial {p}}{\partial {x}} + \frac{\partial {p}}{\partial {y}} \]
